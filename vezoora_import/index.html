Product / Category / Customer Import!

Kindly follow the below steps to import Product through API's

  1. Create SOAP username / password from magento Backend
  2. Update above details in the settings.php file
  3. Update attribute details in the settings.php file
  4. Get the attribute details and options from following link Attribute List
  5. Get the category details from following link Category List
  6. Fill the CSV by appropriate values For Color is Blue means specify 6 in the CSV
  7. Follow the Date Format like yyyy-mm-dd Eg., 2014-06-31
  8. Please don't change the any attribute code of Product, Category and Customer

Product Import

  • Place the CSV file in the following location (source/csv/catalog/) with example file format name is Eg., "product-2014-05-23.csv"
  • Upload the respective product image in the following location Eg., "source/csv/images/2014-05-23" and specify the product name with extension in the CSV (Eg., sample_image.png)
  • Install this extension For importing configurable product Extension
  • Run this script for Product upload Product Import
  • Already sample file is available in this location, (source/csv/catalog/). Kindly rename the file and work.

  • Compulsory Fields Description

    • product_type - Type of the Product (simple,configurable,bundle,virtual,grouped,downloadable)
    • name - Name of the Product
    • description - Description of the Product
    • short_description - Short Description of the Product
    • sku - Sku of the product (unique value)
    • weight - Weight of the Product
    • status - Status of the Product (Check Attribute List)
    • visibility - Visibility of the Product (Check Attribute List)
    • price - Price of the Product
    • tax_class_id - Tax Class of the Product (Check Attribute List)
    • manage_stock - Manage Stock of the Product (0 - No, 1 - Yes)
    • qty - Quantity of the Product
    • is_in_stock - Specify Is in Stock(0) or Out Of Stock(0)
    • category_ids - List of Category Id of the Product // with comma separated (Check Category List)
    • grouped_product_skus - List of Sku which come under main Product // Fill if the product type is grouped
    • associated_skus' - List of Sku which come under Main Product, // Fill if the product type is configurable, simple sku with comma separated
    • configurable_attributes - Fill if the product type is configurable, atribute id with comma separated.
    • option_title1 - Title of the Bundle product // Fill if the product type is bundle
    • option_type1 - // select,radio,checkbox, multi - Specify appropriate selections
    • option_skus1 - List of Skus which come under first title with comma separated
    • option_status1 - // give add for new options or delete for old options
    • option_title2 - Similar to Option1
    • option_type2
    • option_skus2
    • option_status2
    • option_title3 - Similar to Option1
    • option_type3
    • option_skus3
    • option_status3

  • Optional Fields Description

    • news_from_date - New from Date (Format : yyyy-mm-dd Eg., 2014-06-31)
    • news_to_date - New to Date
    • url_key - URL Key of the Product
    • country_of_manufacture - Country of Manufacture (Check Attribute List)
    • manufacturer - Manufacture of the Product
    • color -
    • special_price
    • special_from_date
    • special_to_date
    • meta_title
    • meta_keyword
    • meta_description
    • image1-5 - Image of the Product

Category Import

  • Place the CSV file in the following location (source/csv/catalog/) with example file format name is Eg., "category-2014-05-23.csv"
  • Run this script for Category upload Category Import
  • Already sample file is available in this location, (source/csv/catalog/). Kindly rename the file and work.

  • Compulsory Fields Description

    • parent_category_id - Parent Category of the Category
    • name - Name of the Categoty
    • is_active - Active status of the Category

  • Optional Fields Description

    • custom_design - Specify the Theme settings name
    • custom_apply_to_products //1 or 0
    • custom_design_from - Starting Date of the Custom Design Apply
    • custom_design_to - Ending date of the Custom Design Apply
    • custom_layout_update - Custom XML Layout
    • description - Description of the Category
    • is_anchor - Specify the is anchor is Available //1 or 0
    • meta_description - Meta Description of the Category
    • meta_keywords - Meta Keywords of the Category
    • meta_title - Meta Title of the Category
    • page_layout - Specify the Layout design of the Category Page Check with Attribute list // one column / two column
    • url_key - URL key of the Category
    • include_in_menu - Specify 1 or 0

Customer Import

  • Place the CSV file in the following location (source/csv/customer/) and example file format name is Eg., "customer-2014-05-23.csv"
  • Run this script for Category upload Customer Import
  • Already sample file is available in this location, (source/csv/customer/). Kindly rename the file and work.

  • Compulsory Fields Description

    • email - Customer Email
    • firstname - First Name of the Customer
    • lastname - Last Name of the Customer
    • password - Password of the Customer

  • Optional Fields Description

    • middlename - Middle Name of the Customer
    • prefix - Prefix Name of the Customer
    • suffix - Suffix Name of the Customer
    • dob - Date of Birth of the Customer
    • taxvat - Customer tax/VAT number
    • gender - Customer gender: 1 - Male, 2 - Female
